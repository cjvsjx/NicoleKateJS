Introduction

Nicolekate.js is a JavaScript framework that provides a convenient way to enhance your web development process using shortcodes. It is designed to make your coding faster and more efficient by simplifying common tasks and providing a concise syntax for manipulating HTML elements.

Getting Started

To get started with Nicolekate.js, you need to include the framework in your HTML file. You can either download the framework files and host them locally or include them directly from a CDN (Content Delivery Network) for faster loading.

CDN:

<script src="https://d2vl5w9tpyt79r.cloudfront.net/nk-min.js"></script>

Once the framework is included, you can start using its powerful shortcodes to manipulate HTML elements, apply styles, perform AJAX requests, and more. Refer to the function menu on the left for a list of available shortcodes and their usage examples.

We hope that Nicolekate.js will streamline your web development workflow and help you build amazing websites with ease. Happy coding!

Introduction

NicolekateJs is a JavaScript framework that utilizes shortcodes for faster and more efficient web development.

append

Appends content to the target element(s).

Usage:

[append]appended[".myElement2"][/append]

prepend

Prepends content to the target element(s).

Usage:

[prepend]wow[".myElement3"][/prepend]

insertBefore

Inserts content before the target element(s).

Usage:

[insertBefore][".myElement3"]["#myElement"][/insertBefore]

html

Replaces the HTML content of the target element(s) with new content.

Usage:

[html]Hello, world!["#myElement"][/html]

css

Sets CSS styles for the target element(s).

Usage:

[css]{"color": "red", "font-size": "20px"}["#myElement"][/css]

ajax

Loads content from a URL and inserts it into the target element(s) using AJAX.

Usage:

[ajax]["data.json"]["#myElement"][/ajax]

addClass

Adds a CSS class to the target element(s).

Usage:

[addClass]["new-class"][".myElement3"][/addClass]

click

Triggers a click event on the target element(s).

Usage:

[click]["#myButton"][/click]

autofill

Automatically fills an input field with content.

Usage:

[autofill]Auto-filled content1["#inputbox"][/autofill]
